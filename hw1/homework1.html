  • What is Statistics and its relationship with other disciplines. Difference between Descriptive and Inferential Statistics.


    Statistics is a branch of mathematics that deals with collecting, analizing, and organizing data. In practice, statistics is the idea that we can learn about the properties of large set of objects which are populations and samples It's used in many disciplines like electrical engineering: it's used in the manipulation of random signals, in medicine to analize clinical trials and draw conclusion in scientific studies. It's also used in cybersecurity for risk management and in machine learning.
    To introduce descriptive and Inferential statistics we need to talk about populations and samples:
    • Populations: is a set made of statistical units. Some example may be a class of students, the population of a nation and so long so forth. The choice of a population is on who conducts the study, the only rule to be respected is that the population has to be relevant.
    • Samples: it is simply a sub-set of a population. The sample procedure is pretty hard hence a new branch of statistics has been founded to occupy of this.
    So Statistics is divided in two main branches:
    • Descriptive statistics: it's a term used to practice a descriptive analysis of the data. It's used when we need to present our raw data in a meaningful way
    • Inferential statistics: it's a set of tools that mathematicians use to draw conclusion about the characteristics of a population drawn from the characteristics of a sample and determine how certain can be those.

  • Briefly describe the main sampling methods

  • The main sampling methods are divided in two main categories: probability sampling methods and non probability sampling methods

    Probability sampling methods

    • Simple random sampling: using this way of sampling, every member of the population has the same chance to be included in the sampling frame. To conduct this sampling some random number generators can be used.
    • Systematic sampling: very similar to the previous one. With the only difference that every member is listed with a number and the sample members are chosen at regular intervals. The most important issue is that we have to be sure that there are no hidden patterns.
    • Stratified sampling:Stratified sampling involves dividing the population into subpopulations that may differ in important ways. It allows you draw more precise conclusions by ensuring that every subgroup is properly represented in the sample. To use this sampling method, you divide the population into subgroups (called strata) based on the relevant characteristic. Then calculate the proportions between subgroups to see how many should be chosen for the sample and then apply systematic sampling for the scale.
    • Cluster Sampling: population is divided into subgroups where every subgroup has similar characteristics to the whole sample. Instead of selecting individuals, whole subgroups takes part to the sample randomly.

    Non probability sampling methods

    • Convenience sampling: simply includes individuals who can be the most accessible to the researcher. It's a very inexpensive method but also there's no way to tell if the sample is representative.
    • Voluntary response sampling: it's based on ease of access but the researcher does not choose the participants but they are the ones to volunteer for the research.
    • Purposive sampling: with this method the researcher is asked to put in practice theyr expertise to choose the most adapt sample based on the purpose of the research.
    • Snowball sampling: if the population is hard to access then this method is used to recruit other participant through participants. The downside is the representiveness totally out of control of the researcher.
    • Quota sampling: Quota sampling relies on the non-random selection of a predetermined number or proportion of units. This is called a quota. You first divide the population into mutually exclusive subgroups (called strata) and then recruit sample units until you reach your quota. These units share specific characteristics.
    • (source: https://www.scribbr.com/methodology/sampling-methods/#:~:text=Probability%20sampling%20methods%20include%20simple,a%20chance%20of%20being%20included.)

  • Briefly describe the main experiment designs

  • Completely randomized design: in this design the treatments are assigned to each statistical unit with complete randomness. The downside of this design is the lack of control in the quantity of units to which every treatments is assigned
  • Block randomized design: this design solves the problem of the above one: sample is assigned randomly but this time blocks of fixed size of units are used.
  • Factorial design: factorial experiments are used to draw conclusion about one or more factor or variable. It's called like this because every combination of factors is involved.

    • (source: https://www.britannica.com/science/statistics/Experimental-design )

Write a program in C# or VB.NET that creates a window containing a single line, point, circle, rectangle

                
                        using System;
                        using System.Drawing;
                        using System.Windows.Forms;
                 
                        public class ShapeForm : Form
                        {   
                            public ShapeForm()
                        {
                            this.Size = new Size(800, 800); // Increase the width to accommodate all shapes
                        }
                 
                            protected override void OnPaint(PaintEventArgs e)
                        {
                            base.OnPaint(e);
                            Graphics g = e.Graphics;
                 
                            // Draw a point
                            int centerX = this.ClientSize.Width / 2;
                            int centerY = 50;
                            int r = 10;
                            Pen pen = new Pen(Color.Aquamarine, 2);
                            g.DrawEllipse(pen, centerX - r, centerY - r, 2 * r, 2 * r);
                            pen.Dispose();
                 
                            // Draw a line
                            Point point0 = new Point(100, 100);
                            Point point1 = new Point(700, 100);
                            pen = new Pen(Color.Aquamarine, 2);
                            g.DrawLine(pen, point0, point1);
                            pen.Dispose();
                 
                            // Draw a circle
                            int x0 = this.ClientSize.Width / 2;
                            int y0 = 250;
                            int radius = 100;
                            pen = new Pen(Color.Aquamarine, 2);
                            g.DrawEllipse(pen, x0 - radius, y0 - radius, 2 * radius, 2 * radius);
                            pen.Dispose();
                 
                            // Draw a rectangle
                            int x = 100;
                            int y = 400;
                            int width = 600;
                            int height = 100;
                            pen = new Pen(Color.Aquamarine, 2);
                            g.DrawRectangle(pen, x, y, width, height);
                            pen.Dispose();
                 
                        }
                    }
                 
                    public class Program
                    {
                        public static void Main()
                        {
                            Application.Run(new ShapeForm());
                        }
                    }
